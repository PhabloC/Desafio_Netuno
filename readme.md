# MMORPG NETUDO

Bem-vindo ao MMORPG NETUDO, um jogo de aventura onde você pode criar seu próprio personagem e explorar um vasto mundo cheio de desafios e mistérios.

## Como Jogar

1. Certifique-se de ter o Node.js instalado em seu sistema.
2. Clone este repositório em sua máquina local.
3. Abra o terminal e navegue até o diretório onde você clonou o repositório.
4. Execute o jogo com o seguinte comando:

```bash
node game.js
```

1. Siga as instruções no terminal para criar seu personagem, escolhendo características físicas, classe, ferramenta de batalha e montaria.

## Funcionalidades

1. Criação de personagem personalizado.
2. Escolha entre diferentes classes com atributos únicos.
3. Escolha sua ferramenta de batalha e montaria.
4. Aventure-se em um mundo virtual cheio de desafios.
